How To Use?

  1. Select Grid Size and click submit to Create your canvas
    • Tip: On desktop devices you can create grid upto 130*130 cells under 30 milliseconds,thats 1/33 of a second
  2. Select Block Color from block color Picker
  3. Select Canvas Color from canvas color Picker
  4. Select Border color from border color Picker
  5. Select Pixel size from slider, pixel size ranges from 5 to 20.
  6. To paint on grid, Click on paint brush radio button
  7. To paint single grid do single mouse click
  8. To paint like a brush, double click mouse and drag the mouse on grid to paint multiple cells in one stroke
  9. You can preview the color by hovering the mouse over cell
  10. To erase the pixel art, Click on Eraser radio button
  11. To erase single grid do single mouse click
  12. To erase multiple cells in one stroke, double click mouse and drag the mouse on grid to erase
  13. Click reset to reset Canvas
  14. Click Preview button to preview the pixel art
  15. To download the image, click Download button and save or open the image
